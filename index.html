Real-Time Collaborative Whiteboard

The Real-Time Collaborative Whiteboard is a full-stack web application that allows multiple users to draw simultaneously on a shared whiteboard in real-time. Built using React.js, Node.js, Express.js, Socket.IO, and MongoDB, the project provides seamless communication and synchronization across all connected clients. This project is ideal for online teaching, brainstorming, team meetings, and collaborative sketching — enabling a fluid, live-drawing experience backed by persistent room data storage in MongoDB

The Healthcare Management System is a web-based application built using Django that streamlines patient registration, appointment booking, and medical staff management. It allows patients to register themselves, view their appointments, and cancel them if needed. Admins can manage patients, doctors, and appointments through the built-in Django admin panel. This system simplifies hospital workflow by digitizing core functionalities and improving communication between patients and healthcare providers.
